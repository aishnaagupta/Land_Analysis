# 🌍 Land Analysis

A Flutter-based mobile application designed to assist in land surveying and geographical analysis. The application provides an interactive map interface, user authentication, and tools for analyzing land characteristics such as elevation, distance, and terrain-related information.

---

## 📖 Overview

Land Analysis is a mobile application developed using Flutter that aims to simplify preliminary land surveying and analysis. The application allows users to authenticate, navigate through an interactive map, and access land-related information useful for planning and surveying purposes.

This project was developed as an academic project to explore the integration of GIS technologies, Google Maps, Firebase Authentication, and Flutter.

---

## ✨ Features

- 🔐 Firebase Authentication
  - User Login
  - User Registration

- 🗺 Interactive Google Maps Integration

- 📍 Location-based land visualization

- 📏 Land analysis utilities
  - Elevation Screen
  - Map Screen
  - Distance calculations (planned)
  - Terrain analysis (planned)

- 👤 User-friendly interface

- 📱 Cross-platform Flutter application

## 🚀 Getting Started

### Prerequisites

- Flutter SDK
- Android Studio
- Firebase Project
- Google Maps API Key

---

### Installation

Clone the repository

```bash
git clone https://github.com/aishnaagupta/Land_Analysis.git
```

Go into the project directory

```bash
cd Land_Analysis
```

Install dependencies

```bash
flutter pub get
```

Run the application

```bash
flutter run
```
